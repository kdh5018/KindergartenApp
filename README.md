## 앱 주요 기능
전국에 유치원에 대한 정보를 가져와 사용자들이 원하는 지역을 설정해서 해당하는 지역의 유치원 정보를 조회하고 원하는 유치원을 누르면 그 유치원에 대한 정보를 보다 자세하게 볼 수 있는 앱

## 앱 주요 기술
- 언어: Swift
- UI 프레임워크: UIKit(Storyboard)
- 디자인 패턴: MVVM
- REST API
- Rx
- PageViewController
- Singleton Pattern
